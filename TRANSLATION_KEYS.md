## How to Add Translations to Your System

### Option 1: If using array-based translations

```php
// In your en.php (or similar)
$translations = array_merge($translations, [
    'degree' => 'Degree',
    'salary' => 'Salary',
    // ... add all keys above
]);
```

### Option 2: If using function-based translations

```php
// In your translation function
function translate($key, $lang = 'en') {
    $translations = [
        'en' => [
            'degree' => 'Degree',
            'salary' => 'Salary',
            // ... more keys
        ],
        'ar' => [
            'degree' => 'الدرجة العلمية',
            'salary' => 'الراتب',
            // ... more keys
        ],
        'ku' => [
            'degree' => 'بڕوانامە',
            'salary' => 'مووچە',
            // ... more keys
        ]
    ];

    return $translations[$lang][$key] ?? $key;
}
```

### Option 3: If using JSON files

```json
// en.json
{
  "degree": "Degree",
  "salary": "Salary",
  "filter_degree": "Filter by Degree"
}

// ar.json
{
  "degree": "الدرجة العلمية",
  "salary": "الراتب",
  "filter_degree": "تصفية حسب الدرجة"
}

// ku.json
{
  "degree": "بڕوانامە",
  "salary": "مووچە",
  "filter_degree": "پاڵاوتن بە پێی بڕوانامە"
}
```

## Usage in Code

Replace hardcoded strings with translation function calls:

```php
// Before
<label>Degree</label>

// After
<label><?= translate('degree') ?></label>

// Or if using a different translation system
<label><?= $translations['degree'] ?></label>
```
